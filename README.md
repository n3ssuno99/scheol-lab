# scheol-lab

Scheol Security Lab is a hands-on cybersecurity environment designed to simulate a realistic information system.

It is built around two complementary infrastructures :

### Heaven

A production-like VPS environment exposed to the Internet.

Focus:
* secure service exposure
* TLS automation
* reverse proxy architecture
* system hardening

### Hell

A self-hosted SME-oriented infrastructure deployed sequentially.

Focus:

* network segmentation
* identity management
* detection & response
* automation
* resilience

### Objectives

This lab is designed to develop practical skills in :

* infrastructure security
* detection engineering
* DevSecOps
* incident readiness
